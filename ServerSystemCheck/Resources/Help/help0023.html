Registry Scan

Registry Scans allow you to check for the existence of a registry value and check for correct data.
Entire registry exported .REG files can be also be added for quick scanning.

Instructions

To start adding an Registry Scan resource to your group structure,

  1. Select the group or sub-group that you want to add resources too,
  2. Right-click either:
    1. The group in the left-hand group window, or,
    2. Right-click the right-hand resource windows,
  3. Choose   Add Resource from the pop-up menu,
  4. Choose   Registry Scan from the pop-out menu,
  5. From the new window that appears, you can either:
    1. Fill out the boxes to create a single registry check item, or,
    2. Click the Import REG File button to import an existing *.reg file.

Single Entry

To enter a single registry scan entry, perform the following steps...

  1. Enter the full path of the registry key you want to check for. HKEY_LOCAL_MACHINE is automatically assumed as this is the only key that can be checked remotely.
  2. Enter the name of the registry value you want to check for. Use "(Default)" for the default registry value name
  3. Select the type of registry value you are checking,
    Options are:
        String Value,
        Binary Value,
        DWORD (32-bit) Value,
        QWORD (64-bit) Value,
        Multi-String Value,
        Expandable String Value.
  4. Enter the data of the value you want to check for,
  5. Select the required scan result returned if the current item is not found.
    Options are:
        OK,
        Warning,
        Fail.
  6. Click OK to add the entered resource to the selected group.

You can now continue to add new resources.


Search The Registry

A search button ... is located next to the Registry Key field. This will allow you search for a specific key-value to use.

Clicking the button will open a new window and will automatically display all sub-keys under HKEY_LOCAL_MACHINE.

  1. Drill down the various sub-keys until you find the value you are looking for.
  2. Click the OK button
  3. The window will close, and the correct values will be automatically inserted
  4. Click OK to add the entered resource to the selected group.


Import .REG File

To import an existing exported registry .REG file, perform the following steps...

  1. Click the Import .REG File button,
  2. Click the Select File button,
  3. From the Open File Dialog box, select the file you want to import,
  4. The file will be loaded and all valid entries will be scanned and added to the list window,
  5. Verify the imported entries
  6. Select the required scan result returned if the current items are not found.
    Options are:
        OK,

        Warning,

        Fail.
  7. Click OK to add the entered resources to the selected group.
