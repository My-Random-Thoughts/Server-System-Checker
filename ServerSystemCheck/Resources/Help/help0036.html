Read-Only File Restrictions

When opening a file that is marked as Read-Only by the operating system, several restrictions are put in place.

These are listed below...

You can not...

  •   Add, edit or delete and groups
  •   Add or delete and servers
  •   Add, edit or delete and resources
  •   Change drive exclusions
  •   Fix any potential errors in the SSC configuration document
  •   ...and many, many more.!

You can...

  •   View the properties of groups
  •   View the properties of servers
  •   View the properties of selected resources
  •   Scan servers and perform resource checks
  •   ...and many, many more.!
