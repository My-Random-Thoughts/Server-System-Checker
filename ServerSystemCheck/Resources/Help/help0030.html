Server Connection Editor

The Server connection editor allows you to define connection strings for connecting to servers. These settings are saved per configuration file.

Instructions

To start editing the connections,

  1. Either:
    1. Click the Options menu,
    2. Choose the Configure Server Connections... item
  2. Or:
    1. Right-click a server,
    2. Choose   Properties from the pop-up menu,
    3. From the window that appears, click the Connect... button,
    4. Choose   Configure from the pop-up menu.

From the window that appears, you can add, edit or remove and exiting connections, or replace them all with a default connection list.

Add New Connection

To add a new connection entry to the list,

  1. Click the add button,
  2. A new blank entry will be added to the list, start typing a connection name for this new entry. Press Enter when done
  3. Now enter the connection string for this new connection. Remember to use #server# as the server name substitution

Edit Existing Connection

To edit an existing entry, double click either the name or the connection string to edit the part you need to. Pressing Enter will save the changes, pressing Esc cancel them.

Delete Existing Connection

To delete an existing entry, click the remove button.

Default Connection List

The following table lists the default connection list that is used when pressing the Load Defaults button

    Name Connection String
    HP SMH http://#server#:2301
    HTTP http://#server#
    iDRAC https://#server#:443
    RDP mstsc.exe /v:#server#
