Add Server

Servers are the remote machines that we want to scan.

They can be added by either:

  1. Manually, one at a time,
  2. Importing from a text file, or
  3. Scanning active directory

Instructions

Manually

  1. Select and right-click the group that you want to add servers too,
  2. Choose   Add Server(s) from the pop-up menu,
  3. From the new window that appears
  4. Click the
    5. Add button
  5. Enter the required server name,
  6. Press enter
  7. Repeat the procedure from step 4
  8. Click OK to add the servers to the selected group.


Import From A Text File

  1. Select and right-click the group that you want to add servers too,
  2. Choose   Add Server(s) from the pop-up menu,
  3. From the new window that appears
  4. Select External Text File from the drop down list
  5. Click OK on the information message box
  6. Find and select a text file to import
  7. Click OK to add the servers to the selected group.


Scan Active Directory

  1. Select and right-click the group that you want to add servers too,
  2. Choose   Add Server(s) from the pop-up menu,
  3. From the new window that appears
  4. Select Active Directory from the drop down list
  5. Wait for the OU tree to be created and populated with the servers
  6. Find the OU or server you want to add
    7. Then, either
    • Right-click an OU, and click   Add All Child Items from the pop-up menu,
    • Right-click a server, and click   Add Server from the pop-up menu,
  7. You could also search for the server(s) you want to add by clicking the button
  8. Repeat the procedure from the step above
  9. Click OK to add the servers to the selected group.


Please Note

If you add an IP address, either manually or via an imported text file, the application will try to resolve it into a name using DNS, if the   Automatically resolve IP addresses is selected.
