Group Properties

You can change the name of a group at any time, as well as it's identification colour. The colours are just for your reference, and have no effect on the group itself.

The informational pane at the bottom of the Group Properties window shows some basic statistics about the selected group.

Instructions

To edit the properties of a group,

  1. Right-click the group,
  2. Choose   Properties from the pop-up menu,
  3. From the window that appears, you can see three tabs...
    1. Properties,
    2. Resources, and
    3. Server Details
  4. Click OK once you have finished editing this selected group.

Properties

From this tab you can edit the main properties of the group:

  1. Rename the group to something else,
  2. Choose a colour for this group.
    3. By default, a group colour is yellow, however there are several colours to choose from when creating or editing a group.
    If coloured groups is currently disabled, a button will appear allowing you to enable them.
  3. Enter a description for this group

Resources

This tab lists all the resources assigned to this group and its sub-groups.
It also shows the number of sub-groups and servers within this group.

Server Details

You can gather some system information from every server within this group, and any child groups.

To do this, click the Get Server Details button.
