Running As Administrator

While not required, running this application with administrator privileges allows a few more options. The following list shows the options that are available.

Administrator Options

  1. Scanning software hotfixes and patches.
    Normal security updates and hotfix patches will be scanned,
  2. Scanning the Security eventlog of any server.
    Application and System logs will still be scanned,
  3. Enabling the option to stop and start services that are not in their required/checked state,
  4. Servers with UAC (User Account Control) will block access to certain functions,
    • Registry Searching (scanning is unaffected),
  5. Scanning of any Windows 2012 or later servers, this is due to an enhanced security model on these servers.
