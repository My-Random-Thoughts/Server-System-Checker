Multi-Select Servers

Multi-Select allows you to move or copy as many servers as you need from any number of groups into another single group.

Dragging and dropping servers from one folder to another is fine for the odd server, however if you want to move more than a few servers, this can become tiresome.

Instructions

To multi-select servers,

  1. Right-click any server in the treeview list,
  2. Choose   Multi-Select from the pop-up menu,
  3. From the window that appears, tick any servers you want to move or copy
    1. Either by clicking on the checkboxes to the right of the server names, or
    2. Highlight a server and press the space bar
  4. Click Next  > when you are done
  5. From the list of groups, select the desination group for the selected servers
  6. Optionally, check the   Copy selected servers checkbox.
  7. Click Finish to move or copy the selected servers to the selected group
