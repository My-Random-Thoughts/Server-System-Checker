Server Statistics

You can show the statistics of all the server information you have collected, either from the current group, or for the entire configuration file.

Instructions

You can get to the server statistics window in two different ways...

Method One (entire configuration file stats)

  1. Click the main menu item Search,
  2. Then Show Server Stats


Method Two (selected group, or entire configuration)

  1. Right-click the group you want to check,
  2. Choose   Properties from the pop-up menu,
  3. From the window that appears, select the Server Details tab,
  4. Click the Server Stats button
    Note, this button will be disabled if the selected group has no servers with properties

List By Options

Once the Server Statistics window has opened, it will list all the different operating systems used by your servers.

There are four list options to choose from...

  1. Operating System
  2. Hardware Model
  3. Memory Size
  4. Processor Type
Operating System will also list your servers by their service pack level,
Hardware Model will also list your server by their hardware model type
