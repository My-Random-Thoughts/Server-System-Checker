Get All Server Properties

You can collect the server properties for all the servers below the selected group.

Instructions

Start by getting the properties of a group,

  1. Right-click the group,
  2. Choose   Properties from the pop-up menu,
  3. From the window that appears, select the Server Details tab,
  4. Click Get Server Details,
  5. Wait for the process to complete, monitoring the results of each scan,
  6. Click Close to close the All Server Properties window,
  7. Click OK to close the Group Properties window.
