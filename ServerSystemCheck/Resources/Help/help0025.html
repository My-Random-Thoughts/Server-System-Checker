EventLog Filters

When viewing the detailed information for a server scan result, the eventlog tab contains an option to allow specific entries to be quickly added or removed from the scan resource.

The selected entries will be immediately removed from the list, but the overall change will not take effect until the next scan.

Instructions

  1. From the Eventlog Scans tab, right click an entry to be excluded from future scans.
  2. Choose either...
        Add To Excluded List

        Remove From Specific List
    ...from the pop-up menu.
    Only the relevant option will be available
  3. The selected event will be removed, along with any other events with the same Event ID number.

Show Scan Settings

To see the current list of scan exclusions and inclusions, click the Show Scan Settings button.
