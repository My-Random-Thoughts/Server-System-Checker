Add Group

Groups are used for organising your servers into a logical and structured way. Groups help to organise servers into units so that different resource checks can be set against different groups

By default, a groups colour is yellow, however there are several different colours to choose from when creating or editing a group. See the section titled Group Properties for more information.

Instructions

To start creating your groups,

  1. Right-click anywhere inside the left hand panel,
  2. Choose   Add Group from the pop-up menu,
  3. Enter a name for this new group.

You can now continue to add new groups, or sub-groups until your structure is complete.

Please Note

With a group you can either create a number of sub-groups or had servers listed. Not both.
