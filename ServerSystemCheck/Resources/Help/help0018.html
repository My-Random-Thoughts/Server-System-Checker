Detailed Scan Report

The detailed scan report will give you the complete picture of any errors or warnings that may have occurred when scanning the selected server. This window has several tabs along the top, one for each scan area.

Tab Details

Windows Services

This tab shows any windows services that are not as expected. That is, any stopped service that should be started, and any started service that should be stopped.

Hotfix Patches

This tab shows any hotfixes that are not as expected. That is, any missing hotfixes that should be installed, and any installed hotfixes that shouldn't be there.

Eventlog Scan

This tab shows any eventlog entries that have appeared since the selected server was last rebooted.

Registry Scan

This tab shows any registry entries that are incorrect or missing.

File Check

This tab shows the results of the file scanning check.

WMI Query

This tab shows the results of the WMI queries that you have run

Freespace Threshold

This tab shows the selected servers main system drive (usually C:) as a pie chart representing the free space available. It also lists any other local drives to this server, and the free space available.
