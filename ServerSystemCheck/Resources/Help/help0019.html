Export Result Summary

This option allows you to export the current scan results to a HTML based report for saving and viewing at a later date.

To see an example of the reports producted, click the Show report examples link.

Instructions

To create a new HTML based report,

  1. Report Type :
      2.   Simple (default), or,
        Detailed

  2. Options :
      3.   Include image icons (increases report file size)
        Include server uptime (if available)

  3. If you chose the detailed report, you have the following further options:
    Detailed :
      4.   List all incorrect services
        List all incorrect hotfixes
        List every reported eventlog entry
        List all missing registry keys
        List every incorrect file
        List all WMI query results
        Show system drive space status
           List all scanned drives
              Add excluded drives

  4. Click OK to run the report and save it to the current folder.

Notes

Including image icons into the report will increase the size of the resulting report file, but will give a nicer looking and more professional output.
