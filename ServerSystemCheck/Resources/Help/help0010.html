Freespace Threshold

Freespace Thresholds allow you to set a desired percentage value for the available free drive space. Default values for Critical and Warning are set at 10% and 20%.

Instructions

To start adding a Freespace Threshold resource to your group structure,

  1. Select the group or sub-group that you want to add resources too,
  2. Right-click either:
    1. The group in the left-hand group window, or,
    2. Right-click the right-hand resource windows,
  3. Choose   Add Resource from the pop-up menu,
  4. Choose   Freespace Threshold from the pop-out menu,
  5. From the new window that appears, complete the required details,
    1. Set the Critical value to a percentage between 1% and less than the Warning value,
    2. Set the Warning value to a percentage between 100% and more than the Critical value,
    3. Set the option to   Include or   Exclude non-system drives in the scan results overview.
    Optionally: Click the Reset button to set the default values.
      Default values for Critical is 10%, and for Warning is 20%.
      Default value for Scan Results For Non-System Drives is Exclude
  6. Click OK to add the entered resources to the selected group.

You can now continue to add new resources.
