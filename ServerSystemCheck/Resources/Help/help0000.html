Introduction

Welcome to the Server System Checker application.
This application will help you scan a group of remote servers in several key areas:

  •   Eventlog Scanning
  •   File Checking
  •   Freespace Thresholds
  •   Hotfix Patches
  •   Registry Scanning
  •   WMI Queries
  •   Windows Services

Each area is covered in detail on other help pages.

If you find any bugs, issues, or want to give feedback - including additional functionality, please contact me via email.
