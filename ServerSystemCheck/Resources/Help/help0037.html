WMI Query Examples

WMI Queries allow you to create custom queries to interrogate your servers hardware and software information.

WMI Query Examples

These examples are very simple ones, but more complicated queries can be generated.

Example 1

This example will get the number of logical drives in the server that are formatted as FAT32...

  • SELECT Format FROM Win32_Volume WHERE Format = 'FAT32'
    Return a count value
You can then check against this return value to make sure it is what you require.

Example 2

This example will return either '32-bit' or '64-bit'...

  • SELECT OSArchitecture FROM Win32_OperatingSystem
Again, this can be checked against to weed out 32-bit servers.

Example 3

This will check to see if the selected server is a virtual one from VMware....

  • SELECT Vendor FROM Win32_ComputerSystemProduct WHERE Vendor = 'VMware Inc.'
