Hotfix Patch Check

Hotfixes and Patches that have been applied via Windows Updates can be scanned for, to make sure a defined list of updates have been installed. Optionally you can check for updates that should not be installed too.

They can be added by either:

  1. Manually, one at a time,
  2. Importing from a text file, or
  3. Scanning an existing server

Instructions

To start adding a Hotfix Patch Check resources to your group structure,

  1. Select the group or sub-group that you want to add resources too,
  2. Right-click either:
    1. The group in the left-hand group window, or,
    2. Right-click the right-hand resource windows,
  3. Choose   Add Resource from the pop-up menu,
  4. Choose   Hotfix Patch Check from the pop-out menu,
  5. From the new window that appears, either...

Manually

  1. Click the   Add button
  2. Enter the required hotfix knowledge base code (KBxxxxxx),
  3. Press enter
  4. Repeat the procedure from step 6
  5. Click OK to add the entered resources to the selected group.


Import From Text File

  1. Select External Text File from the drop down list
  2. Click OK on the information message box
  3. Find and select a text file to import
  4. Click OK to add the entered resources to the selected group.


Scanning An Existing Server

  1. Select one of the listed servers from the drop down list
  2. Wait while the list of installed hotfixes is retrieved
  3. Find and select a hotfixes you want to add
  4. Click OK to add the entered resources to the selected group.


You can now continue to add new resources.
