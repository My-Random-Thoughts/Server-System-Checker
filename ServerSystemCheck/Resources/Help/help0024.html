Resource Duplication

When adding any type of resource, it will be checked against the existing list of resources for that particular group hierarchy. If a duplication is found, a window will appear alerting you to the existing resource entry.

Resource Limitations

The following limitations are set for each resource...

  •  Eventlog Scan
    Only one of each type of eventlog scan can be created per group-tree/branch. (application, system, security)

  •   File Check
    Only one scan per file, per group-tree

  •   Free Space Threshold
    Only one scan per group-tree

  •   Hotfix Patch Check
    Only one scan per hotfix, per group-tree

  •   Registry Scan
    Only one Key/Value pair per group-tree

  •   WMI Query
    Only one scan per WMI Class, per group-tree

  •   Windows Services
    Only one scan per service, per group-tree
