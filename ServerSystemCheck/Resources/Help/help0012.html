Server Properties

The Server Properties window gives you a basic overview of the server hardware and software. It's not meant as a replacement for your own cataloguing system, but a quick reference guide.

Instructions

To view the properties of a server,

  1. Right-click the server,
  2. Choose   Properties from the pop-up menu,
  3. From the window that appears, you can:
    1. View the properties of the selected server,
    2. Refresh the properties by clicking the Refresh Data button.
    3. Select any server drives from not being scanned for by clicking the Exclude Drives button.
  4. Click Close to close the Server Properties window.
