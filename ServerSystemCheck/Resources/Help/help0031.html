Windows Themeing Engine

Most of the time this application is run from a general purpose terminal server, or a maintenance or tooling server and as such does not run the Windows themeing engine (or Desktop Experience feature). This feature generally takes up far too much CPU and memory requirements to use in a server environment.

If this feature is enabled however, several enhancements are enabled. The look at feel of the application is much nicer, as you would expect. This application takes advantage of these extra visuals and will display extra functionality that is not possible with out them.
