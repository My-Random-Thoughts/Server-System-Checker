Eventlog Scan

Application, System and Security Eventlogs can be scanned for all new entries since the server was last rebooted.

Instructions

To start adding an Eventlog Scan resource to your group structure,

  1. Select the group or sub-group that you want to add resources too,
  2. Right-click either:
    1. The group in the left-hand group window, or,
    2. Right-click the right-hand resource windows,
  3. Choose   Add Resource from the pop-up menu,
  4. Choose   Eventlog Scan from the pop-out menu,
  5. From the new window that appears, select either:
    1.   Application
    2.   System
    3.   Security
  • Select one of the following two options...

      • Scan All Events or Scan For Specific IDs

    Scan All Events

  • Select one or more of the following eventlog entry types:
    1.     Critical
    2.     Error
    3.     Warning
    4.     Information
    5.     Success Audit
    6.     Failure Audit
  • Enter any exclusions you may have,
  • Click OK to add the entered resources to the selected group.

    • Scan For Specific IDs

    Only the Event IDs listed below will be checked. Any events matching this list will be flagged as a fail.

  • Enter any Event IDs you want to scan specifically for,
  • Click OK to add the entered resources to the selected group.

    • You can now continue to add new resources.

    Sorting And Consolidation Example

    Any Event IDs entered into either the "Exclude" or "Specific" fields will be automatically sorted and consolidated. This helps will readability and allows for quick searching.

    An example of this is shown below...

    • Original:
        9-12, 7, 15, 2-5, 17, 11-20
    • Consolidated:
        2-5, 7, 9-20

    You can see that 6 entries have been reduced down to just 3, the ranges have also been sorted to make readability easier.
    Another example, with the steps taken:

    • Original:
        3-4, 9, 1-2, 8, 12-18, 10, 5-6
    • Sort:
        1-2, 3-4, 5-6, 8, 9, 10, 12-18
    • Merge:
        1-4, 5-6, 8, 9, 10, 12-18
    • Merge:
        1-6, 8, 9, 10, 12-18
    • Merge:
        1-6, 8-10, 12-18
    • Consolidated:
        1-6, 8-10, 12-18

    Again, you can see that all the entries have been reduced down to a simpler format
