Windows Services

Windows Services can be scanned to check if a specified service is started or stopped. The selectable services list is loaded from the localhost first, but can be obtained from the servers added to the selected group.

Instructions

To start adding a Windows Services resource to your group structure,

  1. Select the group or sub-group that you want to add resources too,
  2. Right-click either:
    1. The group in the left-hand group window, or,
    2. Right-click the right-hand resource windows,
  3. Choose   Add Resource from the pop-up menu,
  4. Choose   Windows Services from the pop-out menu,
  5. From the new window that appears, select either one single service or multiple services by holding the CTRL key.
  6. If you want to load the list of services from another server, select it from the drop down list
  7. Select the required scan result returned if the services are not found.
    Options are:
        OK (Ignore),
        Warning,
        Fail.
  8. Click OK to add the entered resources to the selected group.

You can now continue to add new resources.

Notes

Select scan result for missing services allows you to check if a particular service is not installed on a server. If you set this to   Warning or   Fail, then that result will be returned when the server is scanned. If you just want to skip the fact that a service is not installed, set this to   OK (Ignore).
