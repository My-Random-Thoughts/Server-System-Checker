Change State

With the following resource types, it is possible to quickly change the checking state of a particular resource.

      Windows Services

      Hotfix Patches

Instructions

To do this,

  1. Right-click the resource to want to change,
  2. Choose   Change Checking State from the pop-up menu,
  3. The checking state will change.

Check State Toggle Options

If you selected more than one resource to change then the following rules will be applied...

  1. For    Windows Services, the state will toggle between
      2. Running
      Stopped
  2. For    Hotfix Patches, the state will toggle between
      3. Installed
      Not Installed

If select just one service, a window will appear allowing you to change not just the checking state, but also the service missing option

  1. Service Checking State:
        Running,
        Stopped
  2. Scan Result When Missing:
        OK (Ignore),
        Warning,
        Fail
