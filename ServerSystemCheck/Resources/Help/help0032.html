Command Line Options

The following table lists all the command line options for this application, and describes their meanings.

If no command line is given, or just a filename is added with no switches, the application will launch and open the specified file. This is similar to launching the application by double-clicking on an associated file (.SSC).

Commands

    Command Meaning
    -? Shows this help window
    -file:"filename" Specifies the configuration file to load
    File names should be in double-quotes
    -group:"group-path" Specifies the full group path to scan
    Group names should be in double-quotes
    -report:[full|basic] Specifies which report type to run when the scan process has completed

Examples

  1. ssc.exe -file:"c:\all servers.ssc" -group:"Domain Controllers" -report:basic
  2. ssc.exe -file:"c:\all servers.ssc" -group:"Domain Controllers\RODC\Leeds" -report:full
  3. ssc.exe -file:"c:\all servers.ssc" -group:"Servers\Remote\Terminal Servers" -report:full
