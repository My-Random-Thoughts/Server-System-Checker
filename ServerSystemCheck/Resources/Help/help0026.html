File Check

Files can be scanned for on the servers, and can be checked to see if they exist or not, or even the file version or last modified date.

Instructions

To start adding an File Check resource to your group structure,

  1. Select the group or sub-group that you want to add resources too,
  2. Right-click either:
    1. The group in the left-hand group window, or,
    2. Right-click the right-hand resource windows,
  3. Choose   Add Resource from the pop-up menu,
  4. Choose   File Check from the pop-out menu,
  5. From the new window that appears, either:
    1. Type in the location of the file to be checked, or,
    2. Click the ...browse button to select a file.
  6. Select one of the following options:
    1.   Check File Exists
    2.   Check File Does Not Exists
    3.   Modified Date
        4. Enter the required date, then select either of the following options
          On Or Before
          On Or After
    4.   Version
        5. Enter the version you want to check for, then select one of the following options
          Exactly
          Equal Or Less Than
          Equal Or Greater Than
  7. Click OK to add the entered resources to the selected group.

You can now continue to add new resources.

Drive Letters

Please note that drive letters will not be changed from the values entered.

This means, for example, if a file check if configure to look for a file on a E: drive, but the scanned server does not have an E: drive, then the scan will fail.
It is common for older Citrix Presentation Server (pre-XenApp) Servers to use M: as their C: drive.
